# HyreNet QA Automation Project

## Project Overview
This project is created as part of the **HyreNet BugAThon**.  
The objective is to identify and report bugs in the HyreNet AI-powered assessment platform using **manual testing and Selenium automation**.

The project covers:
- Functional testing
- UI validation
- Positive & negative test scenarios
- Automation using Selenium WebDriver
